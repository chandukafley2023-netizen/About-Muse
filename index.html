Development Process

A detailed look at how MUSE was built, the tools used, and the lessons learned.

1. Project Idea & Purpose

MUSE was created to solve a specific problem in the digital art world: generic social media platforms compress image quality and prioritize viral content over artistic merit. The purpose of this project is to create a "sanctuary for creativity"—a web app where artists can share high-fidelity work(still working on to make it better), receive meaningful critiques, and connect with a community that values art first.

2. Tools & Tech

  • HTML5(mostly)/CSS3 Core structure and styling of the application.
  • JavaScript Used for dynamic page interactions and logic. Firebase connection through API key
  • Firebase Backend services for Authentication (Login/Signup).
  • Custom CSS Utility-first framework for responsive design.

3. LLM Usage

AI tools were used as assistants to speed up development and solve logic errors.

Tools Used: Gemini (Code Gen/Fixes), Perplexity (Explanations), Copilot(Escorting with Github features).
Example Prompt: "to the log in page also add like a animation when you hover over the button because it looks like it doesn't work"

4. Changes & Debugging

The Pivot

Initially, the project was a static website ("The Art of Painting") with 5 separate HTML files. It became unmanageable with over 800+ lines of code per file and it also became hard to find errors in the codes. I pivoted to a structured web app design. The new design has been a big step in improvement and is something I feel is more pleasing to me.

Debugging with AI

While AI helped fix syntax errors, it often broke the responsiveness of the layout. My debugging process involved manually reviewing AI-generated code and adjusting CSS classes to ensure the mobile view remained consistent. This has taugth me not to be fully depending on AI to fix the errors.

5. Lessons Learned

  • Scalability Matters: Writing 1000 lines of code in one file is hard to maintain. Breaking things down is essential.
  • AI is a Tool, Not a Creator: I shouldn't have blindly copy-paste AI code at the begining of the journey; you must understand it to fix the layout issues it creates. I have learned to be be the user of AI instead of being used by AI.
  • User Experience: Even small details like button hover animations (added via prompt) significantly improve how the app "feels" to the user. So I have given a lot of time to improve the UI/UX for better interaction.

Project Report

Chandan Kafley Grade 9 Technology Group 2

01 Executive Summary

MUSE is a social media application designed with a goal of spreading traditional art around the global community of artists, creators, and art enthusiasts. Unlike generic social platforms, MUSE prioritizes high-fidelity visual presentation, copyright protection, and niche community engagement. The platform serves as a digital gallery where users can curate portfolios, network with peers, and discover inspiration across various artistic mediums.

02 Introduction

Background

In the current digital era, general-purpose social media platforms often compress image quality and prioritize algorithmic engagement over artistic appreciation. Artists struggle to find dedicated spaces where their work is viewed as art rather than just "content."

Objective

  • Provide high-quality images without aggressive compression.
  • Facilitate meaningful critique and feedback.
  • Connect artists with potential clients.

03 Target Audience

Primary

Digital artists, traditional painters, illustrators, photographers, and sculptors seeking a professional portfolio space.

Secondary

Art collectors, gallery curators, and art enthusiasts looking for inspiration or talent to hire.

04 Future Scope

Marketplace

Sell prints & digital assets directly.

Live Streaming

"Work in Progress" streams.

AR Galleries

Visualize art on walls.

05 Conclusion

MUSE is more than just an app; it is a dedicated ecosystem for the creative mind. By focusing on the specific needs of artists—quality, credit, and community—MUSE addresses the gaps left by mainstream social media, positioning itself as the premier digital destination for art.
